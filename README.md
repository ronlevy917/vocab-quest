# Vocab Quest — upgraded with sentence completion

This package is ready for GitHub Pages.
Upload all files in this ZIP to the root of the repository.

Version: sentences-2026-06-16-v1

New in this version:
- Sentence completion mode: choose the correct vocabulary word to complete an English sentence.
- Works on desktop and iPhone.
- Keeps the previous flashcards, multiple-choice, typing, matching, word list, exam, progress reset and optional Firebase sync.

Important:
- Do not paste Firebase rules into index.html.
- If you use Firebase, copy rules from FIREBASE_RULES_COPY_TO_FIREBASE.txt into Firebase Console > Firestore Database > Rules.
