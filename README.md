# Vocab Quest - גרסה יציבה ובטוחה

הקובץ הראשי של המשחק הוא `index.html` בלבד. הוא חייב להיות בשורש הריפו ב-GitHub Pages.

הגרסה הזאת לא כוללת `firebase-config.js` ולא כוללת `firestore.rules`, כדי שלא יהיה בלבול והאתר לא יציג קוד במקום משחק.

העלאה מומלצת:
1. מחק את כל הקבצים הישנים מהריפו.
2. העלה את כל הקבצים מתוך ה-ZIP הזה לשורש הריפו.
3. ודא ש-`index.html` מתחיל ב-`<!doctype html>`.
4. פתח את האתר עם פרמטר cache חדש, למשל `?v=safe-onefile-2026-06-16-v1`.

מה יש במשחק:
- כרטיסיות
- בחירה מרובה
- הקלדה
- התאמה
- רשימת מילים
- מבחן אמיתי
- השלמת משפטים עם רמז שמופיע רק אחרי לחיצה
- אפשרויות בחירה דומות ומבלבלות
- שמירת התקדמות מקומית
- כפתור איפוס התקדמות
